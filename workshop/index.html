üîë Key Concepts You'll Master

AI Agents

An AI agent is an intelligent software entity that can perceive its environment, make decisions, and take actions to achieve specific goals. In this workshop, your agent will:

  • Understand natural language input from users
  • Decide when to call specific functions (tools)
  • Generate human-like responses

Function Calling (Tools)

Function calling allows AI agents to use external tools. For example, when a user asks "What's the weather in Seattle?", the agent:

  1. Recognizes this needs weather data
  2. Calls the get_weather() function with "Seattle" as parameter
  3. Gets the result and formulates a natural response

Streaming vs Non-Streaming

Non-Streaming

Complete response at once

Wait for entire response, then display. Good for batch processing.

Streaming

Response in real-time chunks

Display response as it's generated. Better user experience.
