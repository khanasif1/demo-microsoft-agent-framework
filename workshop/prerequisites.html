Install and verify required tools

üì¶ What We'll Install

Before we can build our AI agent, we need to set up our development environment. In this module, you'll install and verify all the necessary tools.

üêç Step 1: Install Python 3.8 or Higher

Why Python?

Python is the programming language we'll use to build our AI agent. The Microsoft Agent Framework requires Python 3.8 or higher.

Installation Instructions

ü™ü Windows:
  1. Download Python

    Visit python.org/downloads and download the latest Python 3.x version (3.8 or higher).

  2. Run the Installer

    ‚ö†Ô∏è IMPORTANT: Check the box that says "Add Python to PATH" before clicking Install.

    ‚ö†Ô∏è This is crucial! Without adding Python to PATH, you won't be able to run Python from the command line.
  3. Complete Installation

    Click "Install Now" and wait for the installation to complete.

üçé macOS:

Python comes pre-installed on macOS, but it's usually an older version. Install the latest version using Homebrew:

Terminal
# Install Homebrew if you don't have it
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Python
brew install python@3.11
üêß Linux:
Terminal
# Ubuntu/Debian
sudo apt update
sudo apt install python3.11 python3.11-venv python3-pip

# Fedora
sudo dnf install python3.11

‚úÖ Verify Python Installation

Open a new terminal/command prompt and run:

Command
python --version

You should see output like:

Python 3.11.5
‚úÖ Success! If you see a version number 3.8 or higher, Python is correctly installed.
‚ùå Troubleshooting: If you get "command not found" or "not recognized":
  • On Windows: Try py --version instead
  • On macOS/Linux: Try python3 --version
  • Make sure you checked "Add Python to PATH" during installation
  • Restart your terminal after installation

‚òÅÔ∏è Step 2: Install Azure CLI

Why Azure CLI?

Azure CLI is a command-line tool that lets you interact with Azure services. We'll use it for authentication and managing Azure resources.

Installation Instructions

ü™ü Windows:

Option 1: Using Windows Package Manager (Recommended)

PowerShell
winget install -e --id Microsoft.AzureCLI

Option 2: Download MSI Installer

Visit aka.ms/installazurecliwindows and download the MSI installer.

üçé macOS:
Terminal
brew update && brew install azure-cli
üêß Linux:
Terminal
curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash

‚úÖ Verify Azure CLI Installation

Open a new terminal/command prompt and run:

Command
az --version

You should see output showing the Azure CLI version and Python location:

azure-cli                         2.55.0
core                              2.55.0
telemetry                          1.1.0
Python location 'C:\Program Files\...'
‚úÖ Success! Azure CLI is correctly installed.

üìù Step 3: Install a Text Editor or IDE

You'll need a text editor to view and potentially modify the code. We recommend Visual Studio Code (VS Code) as it's free, powerful, and works on all platforms.

Install VS Code (Recommended)

  • Download VS Code

    Visit code.visualstudio.com and download the installer for your operating system.

  • Install Python Extension

    After installing VS Code, open it and install the Python extension by Microsoft from the Extensions marketplace.

üí° Alternatives: You can also use PyCharm, Sublime Text, Atom, or even a simple text editor like Notepad++ (Windows) or TextEdit (macOS).

üåê Step 4: Azure Account Setup

Create an Azure Account

If you don't have an Azure account yet, you'll need to create one. Microsoft offers a free trial with $200 credit for 30 days.

  1. Sign Up

    Visit azure.microsoft.com/free and click "Start free".

  2. Provide Information

    You'll need to provide:

    • A Microsoft account (or create a new one)
    • Phone number for verification
    • Credit card (for identity verification - you won't be charged unless you upgrade)
  3. Complete Setup

    Follow the on-screen instructions to complete your account setup.

üí∞ Cost Awareness: This workshop will use Azure AI services. With the free trial, you have $200 credit. The expected cost for this workshop is minimal (~$0.50-$2.00). Always monitor your usage in the Azure portal.

‚úÖ Login to Azure CLI

Once you have an Azure account, authenticate Azure CLI:

Command
az login

This will open your web browser. Log in with your Azure account credentials. After successful login, you'll see your subscription information in the terminal.

‚úÖ Success! If you see your subscription details, you're authenticated with Azure.
